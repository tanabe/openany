# What is this

openany extends open command.

## Setup

```
$ npm i
```

## Run as an executable

```
$ npm install -g
```

## Sample

Sample code below may open GitHub repo page in your default browser.

```
$ openany tanabe/openany
```

## Customize

You can customize src/config.ts to your need.

## Uninstall

```
$ npm uninstall -g
```
